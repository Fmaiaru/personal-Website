PEDIDOS APP

Description

"Pedidos App" is an Android application that will allow you to keep track of your orders. It is connected to a database on an online server, which allows several users to enter and thus: view, edit, add and delete orders. The application also has a login and registration system, as well as a password recovery environment. This project was developed in the Android Studio environment, using the JAVA language and also other technologies such as PHP and SQL to work remotely with the Database
